__MoonBit__ App Runtime + FS + Path

This example runs through App and uses the runtime bridge exposed on window.__MoonBit__. It demonstrates direct __MoonBit__.core.invokeOp(...) calls to the fs and path extensions, with the filesystem flow using MoonBit-style calls such as read_file(path), write_file(path, content), exists(path), and readdir(path).
